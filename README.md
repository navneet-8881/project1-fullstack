# Project1 Full Stack Todo App

A simple full-stack project with an Express backend, SQLite database, and a static frontend.

## Features
- Add new todo items
- Toggle completed state
- Delete todos
- Persistent storage using SQLite

## Setup
1. Open a terminal in `project1-fullstack`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:4000`

## Development
- Use `npm run dev` to start the server with `nodemon`.

## Notes
- The backend API is available at `/api/todos`
- The frontend is served from the `public` folder
